## Setup 👷‍♂️

Clone or download this repo

```bash
$ git clone <URL-OF-THIS-PAGE>
```

Install dependencies

```bash
$ npm install
# Or Yarn
$ yarn install
```

Start development server

```bash
$ npm start
# Or yarn
$ yarn start
```
